## BackEnd del producto de "Supermercado"

### Requisitos
- **Básicos**
  - Node.JS versión > 10.x

- **Dependencias** (a través de  ' npm install ' ):
  - _mongodb_
  - _dotenv_
  - _express_
  - _cors_

  :bangbang: La dependencia _nodedemon_ debe instalarse **globalmente** (en una consola, ' npm instal -g nodemon ' ).
